Process Control Library model element

The Process Control Library encapsulates constructive blocks which allow to define a control loop by interconnecting systems. According to the dynamics of the system implemented, the systems are classified into several types of blocks,

  • continuous
  • discrete
  • event-based
  • hybrid

Usage

To add a process control element to your model, drag one or more of the block icon element to the list of your model elements. The blocks can be interconnected by means of the method setInput(). See the help of each block for more information.
