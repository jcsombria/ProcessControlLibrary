SOD sampler element

SODSampler implements a send-on-delta sampler which can easily be included to an EJS simulation.

The element implements a send-on-delta sampler, which can be directly incorporated to an EJS simulation together with other elements from the process control library.

Usage

To add a SODSampler element to your model, simply drag the icon element to the list of your model elements.

A call to the element method void setDelta set the delta threshold.

The element provides the interface Discrete:

  • int getNumberOfStates() Returns the number of discrete states of the block.
  • int getNumberOfInputs() Returns the number of inputs of the block.
  • int getNumberOfOutputs() Returns the number of outputs of the block.
  • double[] evaluate(double x) Returns a value that should be used to detect an event.
  • void update(double x) Update the state of the block.
Note that the states and input is not stored into the element, but they must be handled by EJS.

Example of use

// Create an integrator
double delta = 0.1;
sod.setDelta(delta);
    	
sod.init(y); // where y is the sampled signal
    	
/* The following code must be inserted into an Event page */
return sod.evaluate(y); // Zero condition

sod.update(y); // Action
