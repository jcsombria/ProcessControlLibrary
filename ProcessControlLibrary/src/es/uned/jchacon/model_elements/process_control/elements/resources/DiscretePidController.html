PID controller element

PID implements a continuous PID controller to be included in an EJS simulation.

The element is a complete implementation of a PID controller, with an anti-windup mechanism and derivative filter, which can be directly incorporated into an ODE page with the interface Continuous.

Usage

To add a PID element to your model, simply drag the icon element to the list of your model elements.

The element provides several methods to incorporate it into a simulation. Here are the most frequently used,

  • int getNumberOfStates();
  • int getNumberOfInputs();
  • int getNumberOfOutputs();
  • double[] getRates();
  • double[] getOutput();

Example of use
