Saturation model element

Saturation implements a block whose output is equal to its input, if the value of the input is within the valid range, otherwise it is limited to the saturation values. The element can be directly incorporated into an EJS simulation together with other elements from the process control library.

Usage

To add a Saturation element to your model, simply drag the icon element to the list of your model elements.

Example of use

// Set the saturation limits
saturation.setRange(uMin, uMax);

/* The saturated output is obtained with the following line of code */
saturation.getOutput();
